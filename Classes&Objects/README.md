# Objektorientiertes Programmieren – Zusammenfassung

Ich weiss, dass **Klassen** Baupläne sind und **Objekte** deren konkrete Instanzen.  
Eine **Klasse** besteht aus **Attributen** (Eigenschaften), **Konstruktoren** (zum Erzeugen von Objekten) und **Methoden** (Funktionen des Objekts).  
**Objekte** werden mit dem Schlüsselwort `new` erzeugt und speichern ihren **Zustand** in den Attributen, die man verändern kann.  
Über **Methodenaufrufe** kann man das Verhalten eines Objekts steuern oder Informationen abrufen.
