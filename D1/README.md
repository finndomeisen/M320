# D1

## Ziel
Dieses Projekt zeigt, wie man in Java Klassen mit Konstruktoren, Methoden und gekapselten Attributen erstellt.  
Es wird auch demonstriert, wie Objekte miteinander kommunizieren und wie sich ihr Zustand verändert.

---

## Lernziele

- Ich kann Klassen mit Attributen, Konstruktoren und Methoden definieren.  
- Ich kann Objekte instanziieren.  
- Ich kann zeigen, wie Objekte über Methoden miteinander kommunizieren.  
- Ich kann Attribute kapseln (private + Getter/Setter).  
- Ich kann zeigen, wie sich der Zustand eines Objekts verändert (durch Methodenaufrufe).

---

## Aufbau
**Klasse:** `Auto`
- Attribute: `farbe`, `geschwindigkeit`
- Konstruktor: setzt Startwerte
- Methoden:
  - `beschleunigen(int wert)`
  - `bremsen(int wert)`
  - `rennenMit(Auto gegner)`  
- Getter/Setter für Kapselung

**Main-Klasse:** erstellt zwei Autos, ruft Methoden auf und zeigt Zustandsänderungen.
