- **Hierarchie im Code mittels Vererbung:**  
  Durch Vererbung kann eine Klasse Eigenschaften und Methoden einer anderen Klasse übernehmen, z. B. class Hund extends Tier.

- **IST-Beziehungen umsetzen:**  
  Eine Unterklasse steht in einer *IST-Ein*–Beziehung zur Oberklasse, z. B. „Ein Hund IST ein Tier“.

- **Interfaces implementieren:**  
  Interfaces definieren Methoden, die eine Klasse implementieren muss, z. B. class Auto implements Fahrbar, wodurch festgelegt wird, dass die Klasse alle Methoden des Interfaces umsetzt.
